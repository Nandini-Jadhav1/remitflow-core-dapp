# 📖 RemitFlow User Guide

## Step 1 — Install Freighter
1. Go to https://freighter.app
2. Add to Chrome → Create wallet
3. Save your seed phrase!

## Step 2 — Switch to Testnet
1. Open Freighter
2. Click network name → Select Testnet

## Step 3 — Get Free XLM
1. Copy your wallet address
2. Go to https://friendbot.stellar.org
3. Paste address → Submit → Get 10,000 XLM

## Step 4 — Connect RemitFlow
1. Open the app
2. Click Connect Freighter Wallet
3. Approve in Freighter popup

## Step 5 — Send Payment
1. Enter recipient address (G... 56 chars)
2. Enter amount
3. Toggle Path Payment for USDC
4. Enable Fee Bump for gasless
5. Click Send → Approve in Freighter
6. See success + transaction hash

## Troubleshooting
- Not connecting → Refresh and retry
- Transaction failed → Check balance
- Invalid address → Must be 56 chars starting with G# 📖 RemitFlow User Guide
